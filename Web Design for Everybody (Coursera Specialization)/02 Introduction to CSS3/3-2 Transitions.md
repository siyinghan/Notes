```css
div {
  color: #000000;
  background: #2db34a;
  text-align: center;
  width: 250px;
  height: 200px;
  border-radius: 6px;
  transition-property: color, width, background, border-radius;
  transition-duration: 0.5s;
  transition-timing-function: linear;
  transition-delay: 0.5s;
}

div:hover {
  color: #ffffff;
  width: 350px;
  background: #2d31b3;
  border-radius: 50%;
}
```
